What is HTML ?

HTML stands for HyperText Markup Language . It's the standard language used to create and structure content on the web. HTML uses a system of "tags" to define the structure of a webpage, such as headings, paragraphs, links, images, tables, and so on.

Meaning of HTML

HyperText → Text with links to other documents (like hyperlinks).
Markup → Special tags that define structure (headings, paragraphs, images, tables). Language → It follows rules (syntax) for writing code that browsers understand.

History of HTML →


  • 1989-1991 → Tim Berners-Lee (at CERN) invented the World Wide Web and the first version of HTML.
    HTML 1.0 was very simple (basic text, links, and images).

  • 1995 → HTML 2.0 standardized, adding forms and basic tables.

  • 1997 → HTML 3.2 introduced support for scripting (JavaScript), applets, and more formatting.

  • 1999 → HTML 4.01 became widely used; focused on separating structure (HTML) from design (CSS).

  • 2008-2014 → HTML5 developed and finalized.

Notes:

  • Introduced video/audio tags, canvas for graphics, and support for modern web apps.
  • This is the version we mostly use today.

    • Key Feature →

  • Markup Language → Uses tags (like <p>, <h1>, <img>) to structure content.
  • Not a programming language → It doesn't "do" logic, it just organizes and presents content.
  • Works with CSS & JavaScript:
    HTML = Structure (skeleton)
    CSS = Style (design, colors, fonts)
    JavaScript = Behavior (interactivity, logic)

    • Key parts of this structure →

  • < !DOCTYPE html >: Tells the browser to expect HTML5.

  • < html >: The root element that contains all the HTML content.

  • <head >: Contains metadata like the title of the page and links to stylesheets.

  • < body >: The visible content of the page (text, images, links, etc.).

    • Here's an example of a basic HTML structure →

    <!DOCTYPE html >
    <html lang="en">
    <head >
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title > Document < /title >
    </head>
    <body>

    <!--This body is written inside this -->

    </body>
    </html>
