Internet

  • What is Internet ?

    • The Internet is a global network of computers and devices that are connected together to share information and communicate.

    At its core, it's a huge system of interconnected networks that uses standardized rules (called protocols, like TCP/IP) so devices can "talk" to each other.

  • Worldwide connection: It links billions of computers, smartphones, servers, and other devices across the globe.

  • Data exchange: It allows information (text, images, videos, files, etc.) to move quickly between devices.

  • Services: It powers things like websites (the World Wide Web), email, social media, messaging apps, online games, cloud storage, and more.

  • Infrastructure: Underneath, it relies on cables (undersea and on land), satellites, cell towers, and data centers.

    • In short term :- The Internet is the backbone that makes modern communication, knowledge-sharing, and online services possible.

  • Types of IP Addresses ?

    • An IP address (Internet Protocol address) is a unique number assigned to every device connected to the Internet (like your phone, computer, or even smart TV). It works like a digital address so devices can find and talk to each other.

    Type of IP address :

    There are two types of IP Address i.e. IPv4 (Internet Protocol version 4) and IPv6 (Internet Protocol version6)

    1. IPv4 (Internet Protocol version 4):
      • Format: 192.168.1.1 (four numbers, each 0-255)
      • Example: 8.8.8.8 (Google's DNS server)
      • About 4.3 billion possible addresses.
    2. IPv6 (Internet Protocol version 6)
      • Format: 2001:0db8:85a3:0000:0000:8a2e:0370:7334 (uses hexadecimal and colons)
      • Created because we were running out of IPv4 addresses.
      • Has an almost unlimited supply of addresses.

    Kinds of IP Addresses :

  • Public IP: The main address your internet provider gives you (visible on the internet).
  • Private IP: Used inside your home or office network (like 192.168.x.x).
  • Static IP: Fixed address that never changes (often used by servers/websites).
  • Dynamic IP: Changes each time you connect to the internet (most common for home users).

  • What is MAC Address ?

    • A MAC address (Media Access Control address) is a unique hardware identifier built into your device's network card (Wi-Fi card, Ethernet card, Bluetooth, etc.).

  • It's like a serial number for your device's network interface.
  • Unlike an IP address (which can change), a MAC address is permanent (though it can be spoofed or changed by software).

    • Format :

    Usually written as 6 pairs of hexadecimal digits (0-9, A-F).
    Example:
    First is :-00:1A:2B:3C:4D:5E
    Second is :-00-1A-2B-3C-4D-5E

    Purpose :

    1. Identifies devices at the hardware level inside a local network (LAN).
    2. Used in Ethernet and Wi-Fi communication.
    3. Helps switches and routers deliver data to the right device.

  • What is a Port Number ?

    • A Port number is like a virtual door on your computer or device that helps identify specific services or applications when data is sent or received over the internet.

    Think of it like this :

    Your IP address = your house address (where to send data).

    Your Port number = the specific room or person inside the house (which program/service should handle the data).

    Format :

    A port number is a 16-bit number (0 to 65,535).

    Always paired with an IP address in the format:
    IP:Port
    Example → 192.168.1.5:80

    Why are Port Number important !

    They allow multiple services to run on the same device without conflict.

    Example: Your computer might browse a website (port 443), send an email (port 25), and download a file via FTP (port 21) — all at the same time.

    Firewalls and routers use them to allow or block traffic.

    Port Range :-

    1. Well-Known Ports (0-1023):-

    80 → HTTP (websites)
    443 → HTTPS (secure websites)
    25 → SMTP (email sending)
    22 → SSH (remote login)

    2. Registered Ports (1024-49151):--

    3306 → MySQL database
    3389 → Remote Desktop (RDP)

    3. Dynamic/Ephemeral Ports (49152-65535)

  • What is DNS ?

    • DNS (Domain Name System) is like the phonebook of the internet.

    It translates human-friendly names (like www.google.com) into IP addresses (like 142.250.190.78) so computers can find each other.

    DNS is the global, distributed system that translates the human-friendly domain names into the computer-friendly IP addresses. Without it, you'd have to memorize hundreds of IP addresses to browse the web.

    How DNS Works (Step by Step) ?

    1. Browser Check : First, your computer checks if it already knows the IP (cached).
    2. DNS Resolver : If not, it asks a DNS resolver (usually your ISP or Google DNS 8.8.8.8).
    3. Root Server : If the resolver doesn't know, it asks a root DNS server.
    4. TLD Server: The root server directs it to the Top-Level Domain (TLD) server (e.g., .com server).
    5. Authoritative Server: That points to the authoritative DNS server for youtube.com.
    6. Get IP Address: The server replies with the real IP (e.g., 142.250.190.78).
    7. Connect : Your browser uses that IP to load YouTube.

    Note :-This whole process usually happens in milliseconds.

    Common Public DNS Servers :-

    Google DNS → 8.8.8.8 and 8.8.4.4
    Cloudflare DNS → 1.1.1.1
    OpenDNS → 208.67.222.222
