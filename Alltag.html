All important Tag of HTML

<HTML> tag :

Purpose: The root element that contains all HTML content.
Usage: All other HTML tags go inside this tag.

<HEAD> tag :

Purpose: Contains metadata for the document (like title, character encoding, stylesheets, etc.).
Usage: Placed within < html > but outside the <body >.

<TITLE> tag :

Purpose: Specifies the title of the webpage that appears in the browser's title bar or tab.
Usage: Always placed inside the tag.

<BODY> tag :

Purpose: Contains the visible content of the webpage (what the user sees).
Usage: Everything the user interacts with, like text, images, and links, goes inside the < body > tag.

<H > tag :

Purpose: Headings that structure your content in a hierarchy. < h1 > is the most important, and < h6 > is the least important.
Usage: Headings are typically used for titles or section titles.

<P> tag :

Purpose: Defines a paragraph of text.
Usage: Used for normal text content.

<A> tag :

Purpose: Creates a hyperlink to another webpage or resource.
Usage: It often uses the href attribute to specify the destination.

<Img> tag :

Purpose: Embeds an image on the page.
Usage: It uses the src attribute to specify the image file and alt for alternative text.

<UL & OL> tag :

Purpose: Used for lists of items. < ul > is for unordered (bulleted) lists, and < ol > is for ordered (numbered) lists.
Usage: Each list item is wrapped in an < li > tag.

<LI> tag :

Purpose : Listing the Name and items.

<Div & Span > tag :

Purpose: < div > is a block-level container used to group content, while < span > is an inline container for small portions of content.
Usage: Useful for styling or organizing content with CSS.

<Table> tag :

Purpose: Defines a table to display data in rows and columns.
Usage: Contains other tags like < tr >, < td >, and < th >.
TR - Represents a row in a table.
TD - Defines a cell in a table that holds data.
TH - Defines a header cell in a table. By default, it is bold and centered.

<Form> tag :

Purpose: Defines an HTML form for user input.
Usage: Contains elements like text fields, buttons, checkboxes, etc.

<Input> tag :

Purpose: Defines an input field in a form (text, checkbox, radio button, etc.).
Usage: Often used inside forms for user interaction.

<Textarea> tag :

Purpose: Creates a multi-line text input area (like a comment box).
Usage: Often used in forms to allow users to type longer text.

<Select> tag :

Purpose: Creates a dropdown menu.
Usage: Contains < option > elements that define the choices.

<Option> tag :

Purpose: Defines options in a dropdown (< select >) menu.
Usage: Placed inside the < select > tag.

<Button> tag :

Purpose: Creates a clickable button (useful for submitting forms or triggering JavaScript actions).
Usage: Can be used for submitting a form or for any other button action.

<Horizontal Rule> tag :

Purpose: Creates a horizontal line (often used as a separator between sections).
Usage: It is self-closing and doesn't need an ending tag.

<Iframe> tag :

Purpose: Embeds an external webpage within your webpage.
Usage: Typically used to embed videos, maps, or other websites.

<META> tag :

Purpose: Provides metadata like the character set, description, and keywords for search engines.
Usage: Typically placed in the <head > section.

<Link> tag :

Purpose: Links external resources, typically used to link to stylesheets (CSS).
Usage: Placed inside the section.

<Script> tag :

Purpose: Embeds JavaScript into the webpage or links to an external script.
Usage: Can be placed inside the < head > or at the end of the < body > for better performance.

<Hr> tag :

Purpose: Creates a horizontal line (often used as a separator between sections).
Usage: It is self-closing and doesn't need an ending tag.
